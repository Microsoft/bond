Detailed Description

template<typename Request, typename Response>
class bond::ext::grpc::unary_call< Request, Response >

Exclusive owner of the details of a single async, unary call.

Call Finish or FinishWithError to send a response back to the client.

If no explicit call to Finish or FinishWithError has been made before this unary_call instance is destroyed or moved from, a generic internal server error is sent.

Note
This class can only be moved. If shared ownership semantics are needed, convert this to a shared_unary_call with the share function.
